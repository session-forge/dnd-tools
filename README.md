# D&D 5e 2024 Tools

A collection of web-based utilities for Dungeon Masters running D&D 5e 2024 campaigns.

## Tools Included

- **Combat Status Tracker** - Track conditions, spells, and weapon mastery effects for all combatants. Pre-plan encounters with drag-and-drop initiative ordering.
- **Dice Roller** - Roll all 7 D&D dice types with modifiers, advantage/disadvantage, and roll history. As truly random as your hardware allows.
- **Combat Scene Generator** - Parse stat blocks and generate tactical combat encounters formatted for Obsidian notes.
- **Encounter Scaler** - Scale existing encounters for different party sizes and levels using 2024 DMG XP budgets.

## How to Use

1. Visit the [live site](https://session-forge.github.io/dnd-tools/)
2. Click any tool to open it

All tools work offline once loaded and save data locally in your browser.

## For DMs

These tools are designed for real at-the-table use with dark themes to avoid screen glare during sessions. The Combat Status Tracker and Dice Roller are optimized for mobile/tablet use.
